# Set of functions that implements general purpose utility class for different scopes

**`Please note that this library currently has no documentation avialable.`**

## Installation

The recommended way to install Support Manager is via Composer:

```bash
composer require biurad/biurad-helpers
```

It requires PHP version 7.1 and supports PHP up to 7.4. The dev-master version requires PHP 7.2.

## Testing

To run the tests you'll have to start the included node based server if any first in a separate terminal window.

With the server running, you can start testing.

```bash
vendor/bin/phpunit
```
